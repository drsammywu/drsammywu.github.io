# Images for the website

## Doctor photo (About page)
- **File:** `dr-wu.jpeg` or `dr-wu.png`
- **Location:** Put the file in this folder (`public/images/`).
- The About page will display it at the top. Use a professional headshot; recommended size at least 400×400 px.

## Experience logos (timeline cards)
- **Folder:** `public/images/experience/`
- **Files:** Add one image per institution. Name them exactly:
  - `burke-logo.png` — Burke Rehabilitation
  - `msk-logo.png` — Memorial Sloan Kettering Cancer Center
  - `nyp-logo.png` — NewYork-Presbyterian Hospital
  - `lenox-hill-logo.png` — Lenox Hill Hospital
- **Format:** PNG or JPG. Logos will be shown at about 48×48 px (will scale to fit).
